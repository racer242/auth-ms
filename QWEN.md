## Qwen Added Memories
- Инструкции по разработке (agent_instructions.txt):

1. При добавлении нового функционала:
   - Проверять, нужны ли настраиваемые параметры и предусматривать их в .env
   - Добавлять юнит-тесты

2. Для каждой итерации разработки:
   - Итерация должна заканчиваться отсутствием ошибок и предупреждений в консоли
   - Проверять прохождение юнит-тестов. При наличии ошибок - исправлять их
   - Обновлять в readme.md описание функционала
   - Заносить в changelog.md краткое описание нового функционала и изменений
   - Сопровождать коммитом в Git. Коммиты Git необходимо называть по-английски

Проверять выполнение этих инструкций после каждой рабочей итерации.
